## 📜 소개 (Introduction)
**공명문(Gongmyungmun)**은 복잡한 코드를 직관적인 **훈민 기호(Hunmin Symbols)**로 변환하여, 고사양 IDE 없이도 논리의 흐름과 시스템의 부하(Entropy)를 시각화하는 프로젝트입니다.

이 프로젝트는 **"메모장(Notepad) 상태에서도 시스템의 자아를 읽을 수 있어야 한다"**는 철학에서 시작되었습니다.

## 🔑 핵심 기호 (The Symbols)
| 기호 | 의미 (Meaning) | 역할 |
|:---:|:---|:---|
| `●` | **Start / Init** | 이야기의 시작, 비트의 생성 |
| `∴` | **Branch / If** | 조건에 따른 분기 ("그러므로") |
| `◎` | **Loop / Cycle** | 반복되는 에너지의 순환 |
| `⇒` | **Transition** | 상태의 전이 및 결과 도출 |
| `×` | **Error / Exception** | 흐름의 단절, 예외 발생 |

## 🛠️ 설치 및 사용 (Usage)
*(추후 공개 예정인 `cerebrum.py` 사용법)*
1. Python 3.x 환경이 필요합니다.
2. 코드를 입력하면 16-bit Field Analysis를 거쳐 공명문으로 변환됩니다.
